发表于 2021-12-14 分类于 tech Disqus：

记录一下自己搭建博客的过程。

一、准备工作

github

首先需要一个github博客， 新建一个名为你的用户名.github.io的仓库，此后的博客文件会部署在这个仓库里。

域名

购买一个自己的域名（或者不购买也是可以的），我是在阿里云上购买的，一块钱可以买一年。

阿里云购买需要进行一系列的实名认证。

安装git和npm

如果之前没有安装过，到网上搜索git官网下载，根据默认的配置安装即可。

安装成功之后，在任一文件夹右键，会出现git bush here的选项，可以打开git命令行窗口。

安装npm也是同理。
